## Brand Foundation

### Visual Positioning
ClickPoint sits at the intersection of **trusted advisor** and **modern expertise**. We're not a cold tech company with robot imagery. We're not a generic business consultancy with handshake stock photos. We're **warm authority**—the smart friend who happens to know everything about AI and genuinely wants to help you figure it out.

**Visual references:** Think Notion's clarity meets Australian financial services warmth. Clean like Linear, but approachable like Xero's brand communications. Never sterile. Never cluttered.

### Master Colour Palette

| Role | Colour Name | Hex Code | Usage |
|------|-------------|----------|-------|
| Primary | Deep Navy | #1A2B4A | Headlines, primary backgrounds, authority moments |
| Secondary | Warm White | #FAF9F7 | Body backgrounds, breathing space |
| Accent | Confident Teal | #2A9D8F | CTAs, highlights, data points, "after" states |
| Supporting | Soft Stone | #E8E4DF | Secondary backgrounds, dividers, subtle cards |
| Supporting | Warm Charcoal | #3D3D3D | Body text, secondary typography |
| Alert/Energy | Sunrise Coral | #E76F51 | Sparingly—key stats, urgent moments, "before" pain states |

### Master Typography System

| Role | Font | Weight | Usage |
|------|------|--------|-------|
| Headlines | **Inter** | Bold (700) | All headlines, key statements |
| Subheads | **Inter** | Semi-Bold (600) | Section headers, carousel titles |
| Body | **Inter** | Regular (400) | Body copy, explanatory text |
| Accent/Data | **Inter** | Medium (500) | Stats, pull quotes, emphasis |
| Alternative Display | **Instrument Serif** | Regular | Optional—hero moments only, warmth injection |

*Note: Inter is chosen for cross-platform consistency (native to Figma, excellent web rendering, free). Instrument Serif adds editorial warmth when we need to feel less "tech deck" and more "trusted publication."*

## Brief 1: LinkedIn Post Template (Text-Heavy Posts)

**Asset Purpose:** Branded background for LinkedIn posts that accompany longer-form written content. These amplify authority and stop the scroll without competing with the post text itself.

**Dimensions:** 1200 x 627px (LinkedIn recommended)

### Layout Specification

```
┌─────────────────────────────────────────────────────────┐
│ [8px padding all sides]                                 │
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │                                                   │   │
│  │     HEADLINE TEXT                                │   │
│  │     [Centred, max 2 lines]                       │   │
│  │                                                   │   │
│  │     ─────────────────                            │   │
│  │     [40px teal accent line, 3px weight]          │   │
│  │                                                   │   │
│  │     Supporting line or URL                       │   │
│  │                                                   │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│  [Bottom left: Logo lockup]    [Bottom right: subtle   │
│                                 geometric accent]       │
└─────────────────────────────────────────────────────────┘
```

### Background Treatment
- **Primary option:** Solid Deep Navy (#1A2B4A) with subtle radial gradient—centre point at 15% opacity Confident Teal (#2A9D8F) fading to edges
- **Secondary option:** Warm White (#FAF9F7) with Deep Navy text for lighter content
- **Texture:** Optional subtle grain overlay at 3–5% opacity for tactile warmth

### Typography on Template
- **Headline:** Inter Bold, 48–56px, Warm White (#FAF9F7) on dark / Deep Navy (#1A2B4A) on light
- **Supporting text:** Inter Regular, 20–24px, 80% opacity of headline colour
- **Accent line:** 40px width, 3px height, Confident Teal (#2A9D8F), centred below headline

### Logo Placement
- Bottom left corner, 16px from edges
- Use white logo version on dark backgrounds
- Size: Logo height approximately 28–32px
- Include wordmark: "ClickPoint" or full "ClickPoint Consulting"

### URL Treatment
- If URL included: Inter Medium, 16px, Confident Teal (#2A9D8F)
- Position: Centred below accent line, or bottom right above geometric accent
- Format: "clickpointconsulting.com.au" (no https://, no www)

### Geometric Accent (Bottom Right)
- Abstract angular shape suggesting forward momentum
- Confident Teal (#2A9D8F) at 20% opacity
- Contained within bottom right quadrant, partially cropped by frame edge
- Should feel intentional but not distracting

### Deliverables
- Figma component with text override fields
- 2x background variants (dark/light)
- Export: PNG, 2x resolution (2400 x 1254px), sRGB colour profile
